Glossary
########

.. glossary::

   AxisArray
      A specialized message format in `ezmsg` for handling multidimensional data arrays.

   ezmsg
      A messaging framework designed for efficient communication and signal processing.

   PyPI
      The **Py**\ thon **P**\ ackage **I**\ ndex, a repository of software for the Python programming language.

   pip
      A package management system for installing and managing Python packages.

   Signal Processing
      The analysis, interpretation, and manipulation of signals using the `ezmsg` framework.

   terminal
      A text-based interface for interacting with the operating system, often used to run commands and scripts. In Windows, it is often referred to as Command Prompt or PowerShell, while in macOS and Linux, it is typically called Terminal.
